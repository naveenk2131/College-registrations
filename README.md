 College Event Registration Page (HTML Only)

This project is a simple "College Event Registration Page" created using pure HTML. 
It is designed to demonstrate a clear understanding of HTML fundamentals such as forms, input elements, tables, and basic page structure.

 Project Overview

The College Event Registration Page allows students to:
- Enter personal details
- Select department and year
- Choose events to participate in
- Provide contact information
- View the event schedule in a table format.
 Technologies Used;
         HTML


 Features

- Student registration form
- Radio buttons and checkboxes
- Dropdown menus
- Text area for additional information
- Event schedule table
- Clean and semantic HTML structure
